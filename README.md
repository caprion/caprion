## What I'm building

### [SpokenKannada](https://github.com/caprion/spokenKannada) — Learn Bangalore Kannada

AI-powered language learning for colloquial Bangalore Kannada. Spaced repetition flashcards, real neighborhood scenarios, and a conversational tutor — all backed by Sarvam.ai.

<img src="https://raw.githubusercontent.com/caprion/spokenKannada/master/screenshots/01-browse-categories.png" width="400" alt="SpokenKannada — browse categories"> <img src="https://raw.githubusercontent.com/caprion/spokenKannada/master/screenshots/06-translate.png" width="400" alt="SpokenKannada — translate">

### [localseek](https://github.com/caprion/localseek) — Local-first document search

Full-text search across your local files. SQLite FTS5, BM25 ranking, zero external dependencies. Optional LLM summarization via Ollama.

<img src="https://raw.githubusercontent.com/caprion/localseek/master/screenshots/01-home.png" width="400" alt="localseek — home"> <img src="https://raw.githubusercontent.com/caprion/localseek/master/screenshots/02-search-results.png" width="400" alt="localseek — search results">

### [Running Analytics](https://github.com/caprion/running) — Garmin sync + training dashboard

Syncs from Garmin Connect, 10-page Streamlit dashboard. Consistency tracking, race confidence, form analysis, recovery monitoring.

<img src="https://raw.githubusercontent.com/caprion/running/master/screenshots/01-overview.png" width="400" alt="Running — overview"> <img src="https://raw.githubusercontent.com/caprion/running/master/screenshots/04-race-confidence.png" width="400" alt="Running — race confidence">

### [airtrail](https://github.com/caprion/airtrail) — Know what you're flying over

Offline PWA that shows mountains, rivers, and cities along your flight path. Pre-load before boarding, use during flight.
